## How to install
+ Open LiveSplit
+ Menu → Edit Layout
+ (+) sign → Control → Scriptable Auto Splitter
+ Double click on "Scriptable Auto Splitter"
+ Browse → Script path
+ In the "advanced" window select the needed paths (don't forget to add them to timer).
